HTML Reference Guide

Basic HTML

Tag Name Description
<!DOCTYPE> Document Type Defines the document type
<html> HTML Tag Defines an HTML document
<head> Head tag Contains metadata/information for the document
<title> Title tag Defines a title for the document
<body> Body tag Defines the document's body
<h1> to <h6> h1 to h6 Heading tags Defines HTML headings
<p> Paragraph tag Defines a paragraph element
<br> Line-break tag Inserts a single line-break
<hr> Horizontal Rule tag Defines a thematic change in content
<!--...--!> Comment tag Defines a comment
