HTML & CSS Cheatsheet

Basic HTML Tags

Tag Name Description
<!DOCTYPE> Document Type Defines the document type
<html> HTML Tag Defines an HTML document
<head> Head tag Contains metadata/information for the document
<title> Title tag Defines a title for the document
<body> Body tag Defines the document's body
<h1> to <h6> h1 to h6 Heading tags Defines HTML headings
<p> Paragraph tag Defines a paragraph element
<br> Line-break tag Inserts a single line-break
<hr> Horizontal Rule tag Defines a thematic change in content
<!--...--!> Comment tag Defines a comment

Basic CSS Selector Types

Selector Type Selector Syntax Description
Universal * Applies declarations to all elements
Class .class-name Applies declarations to all elements with class="class-name"
ID #id-name Applies declarations to a single element with id="id-name"
Element element-name Applies declartions to all elements the specified element type
Pseudo-class selector:condition Applies declarations to all specified elements with the specified condition
Attribute [attribute-name] Applies declarations to all elements with the specified attribute
